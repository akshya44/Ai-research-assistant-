# 🧠 Personal AI Research Assistant

A modern, interactive React application powered by the Google Gemini API. This application acts as your personal AI assistant to help you research topics, organize findings into categories, and chat interactively with your saved research context.

## ✨ Features

- **🎙️ Voice & Text Input**: Ask for research either by typing or by using voice recognition.
- **📚 Categorized Research**: Save your research intuitively under distinct categories (Academic, Professional, Personal, Content).
- **📝 Smart Summarization**: Automatically get clear summaries, key bullet points, and related subtopics generated by Gemini 2.5 Flash.
- **🗺️ Knowledge Map**: Visualize your active research topics as a dynamic word cloud.
- **💬 Context-Aware Chat**: Converse directly with your AI assistant about the topics you've already researched.
- **📊 Real-time Statistics**: See at a glance how many topics, key points, and categories you've explored.

## 🛠️ Tech Stack

- **Frontend Framework**: React 19 + Vite
- **Language**: JavaScript (ESModules)
- **AI Integration**: Google Gemini API (`gemini-2.5-flash`)
- **Styling**: Vanilla CSS (CSS Variables for dynamic themes)

## 🚀 Getting Started

Follow these instructions to get a local copy up and running.

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn
- A Google Gemini API Key. You can get yours at [Google AI Studio](https://aistudio.google.com/).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/akshya44/Ai-research-assistant-.git
   cd "Ai-research-assistant-"
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   - Create a `.env` file in the root directory. You can use the provided `.env.example` as a template.
   - Open `.env` and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_actual_api_key_here
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open the app**
   Visit the localhost URL shown in your terminal (usually `http://localhost:5173`) in your browser.

## 💻 How to Use

1. **Start Researching**: Type a topic into the search bar or click the mic (🎤) to speak.
2. **Select Category**: Pick a relevant category from the dropdown menu (Academic, Professional, Personal, Content).
3. **Save**: Click "Research". The AI will generate a summary and key points, which are automatically saved into your dashboard.
4. **Knowledge Map**: Switch to the **Knowledge Map** tab to visualize the most prominent terms in your research.
5. **Chat**: Switch to the **Chat** tab to ask the AI questions based *only* on the context of your saved research.
